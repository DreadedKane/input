# Input

Unity package for input management.

## Features
* Get controls.
* Get controllers.
* Custom controller for head tracking.

## Requirements
* [Rewired](https://assetstore.unity.com/packages/tools/utilities/rewired-21676)
* [Core](https://github.com/DreadedKane/core)
